Colossus — embed gallery

Each frame below is the same renderer as the full app, loaded chromeless via ?embed=1. A map is fully described by its URL — the view, camera, color-by channel and scale, theme, and any coarse filters — so an <iframe> reproduces it exactly. This page is plain HTML/CSS/JS (no build step) and is the stress test: one baked table rendered many ways.

Add datasets by baking more views and appending to the MAPS array in this file — nothing else changes. Params: view, color, scale, bins, midpoint, scheme, reverse, theme, lng/lat/z, f_<dim>.
